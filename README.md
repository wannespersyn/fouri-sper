# Fouri SPER

Kampplanner voor de fourier: menuplanning, groepen & diëten, activiteiten, leveranciers, voorraad en boodschappenlijsten voor scoutskampen.

## Stack

- Next.js 15 (App Router) + TypeScript + Tailwind CSS
- Supabase (Postgres + Auth + RLS), run locally via the Supabase CLI + Docker

## Development

```bash
supabase start      # start local Supabase (needs Docker running)
npm run dev          # start the Next.js dev server
```
